# Taylor Family Heritage

Individual biographies assembled from FamilySearch records, life histories, and genealogical sources in the Taylor/Naylor/Kofford/Ward family tree of Erik Byron Taylor (b. 1979, Provo, Utah).

## Structure
- `biographies/` — one file per couple, ordered by richness of source material
- `sources/` — raw extracts from GEDCOM

## Approach
Each biography is a standalone text narrative drawn from all available sources: life sketches, obituaries, newspaper clippings, census records, family histories, and personal writings. The goal is a reliable foundation for a comprehensive family narrative.
